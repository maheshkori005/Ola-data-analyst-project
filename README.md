# OLA Ride Booking Data Analysis

## Project Overview
This project focuses on analyzing OLA ride booking data using SQL and Power BI. The goal is to derive meaningful business insights from ride bookings, cancellations, customer behavior, driver performance, and revenue generation.

## Project Architecture

Excel Dataset → SQL Analysis → Power BI Dashboard → Business Insights

## Tools Used
- Microsoft Excel
- MySQL
- Power BI
- SQL

## Dataset Information

The dataset includes:

- Booking ID
- Date & Time
- Booking Status
- Customer ID
- Vehicle Type
- Pickup Location
- Drop Location
- Ride Distance
- Booking Value
- Payment Method
- Driver Ratings
- Customer Ratings
- Cancellation Reasons
- Incomplete Ride Information

## Key Analysis Performed

### Booking Analysis
- Total Bookings
- Successful Bookings
- Cancelled Bookings
- Incomplete Rides

### Revenue Analysis
- Total Revenue
- Average Booking Value
- Revenue by Vehicle Type

### Customer Analysis
- Customer Ratings
- Cancellation Behavior
- Preferred Payment Methods

### Driver Analysis
- Driver Ratings
- Driver Cancellation Reasons
- Performance Evaluation

### Ride Analysis
- Average Ride Distance
- Ride Completion Rate
- Vehicle-wise Ride Distribution

## Dashboard Features

## Business Insights

- Identifies high-performing vehicle categories.
- Highlights major cancellation reasons.
